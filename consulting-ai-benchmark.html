Leaderboard

Key Findings

  • Claude Sonnet 4.5 demonstrates superior performance in nuanced client communication, particularly in handling sensitive topics like project delays and budget concerns.
  • GPT-4 Turbo excels at structured, analytical tasks such as stakeholder analysis and data-driven recommendations, making it ideal for strategic planning work.
  • Context depth matters: Models with larger context windows (200K+) maintained better consistency across complex, multi-stakeholder scenarios.
  • Tone calibration: All tested models required careful prompting to achieve the appropriate formality level for C-suite communications—zero-shot performance varied significantly.

Methodology

Identical Prompts

Each model received the exact same prompt with no modifications, ensuring fair comparison across all evaluations.

Zero-Shot Evaluation

No examples or few-shot learning provided. This tests real-world performance when consultants need immediate responses.

Documented Versions

All model versions and test dates documented for transparency and reproducibility of results.

Real Consulting Tasks

Scenarios derived from actual Big Four consulting engagements, ensuring practical relevance.
